# Sentiment-Analysis-of-US-Airlines-using-Twitter-Data
# Airline Twitter Sentiment Analysis

## Overview
This Google Colab notebook contains the implementation of a social media analytics project focused on analyzing customer sentiment toward major airlines using Twitter data. The project applies data preprocessing, exploratory data analysis (EDA), sentiment aggregation, and visualization techniques to extract business-relevant insights.


## Tools and Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- NLTK / Text preprocessing libraries
- Google Colab environment

## How to Use
1. Open the notebook in Google Colab.
2. Run the cells sequentially from top to bottom.
3. Review generated figures and outputs for insights.

## Notes
- The dataset used is: https://www.kaggle.com/datasets/crowdflower/twitter-airline-sentiment
